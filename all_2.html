Linux下切换用户的命令

  1. 增加用户，在root下：

add user git

表示增加git用户。

  1. 切换用户

su git

切换成git用户。

从root用户切换到其他用户不需要密码，从其他用户切换到root用户，需要输入密码。

ThinkGo类ThinkPHP的Go语言WEB开发框架
